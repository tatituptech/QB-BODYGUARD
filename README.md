TATITUPTECH BODYGUARD
A comprehensive FiveM bodyguard script for QBCore framework that allows players to spawn gang members as bodyguards with automatic weapons who will follow and defend you in combat.

## Features

✅ **5 Different Gang Types**
- Crip (Ballas)
- Blood (Family)
- Vato (Vago)
- Mafia (Mara)
- Lost MC

✅ **Realistic Gang Members**
- Random ped models from each gang
- Authentic gang member appearances
- Proper outfit and styling per gang

✅ **Combat Ready**
- Equipped with automatic weapons (Assault Rifles)
- 500 rounds of ammunition per bodyguard
- Advanced combat AI
- Defensive combat abilities

✅ **Smart Following System**
- Bodyguards follow extremely close to you
- Smooth pathfinding and navigation
- Automatic grouping system
- Never leave group functionality

✅ **Advanced Combat**
- Automatically engages nearby threats
- 30-unit combat range
- High combat ability (100/100)
- Tactical movement patterns

✅ **Easy Management**
- Simple menu system
- Max 5 bodyguards at a time
- One-command dismiss all
- Automatic cleanup on death

## Installation

### Prerequisites
- FiveM Server
- QBCore Framework
- qb-menu resource
